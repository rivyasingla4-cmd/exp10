# AIM

To study the Pandas library in Python and perform basic operations such as creating Series and DataFrames, examining their structure, accessing data, updating values, and modifying columns.

# THEORY

Pandas is a powerful Python library used for data analysis and manipulation. It provides easy-to-use data structures like Series and DataFrame for handling structured data efficiently. Pandas is widely used for data cleaning, exploratory data analysis (EDA), and data processing.

Important Commands Used

import pandas as pd-

Imports the Pandas library and assigns the alias pd.

pd.Series()-

Creates a one-dimensional labeled array capable of holding any data type.

pd.DataFrame()-

Creates a two-dimensional table-like structure with rows and columns.

head()-

Displays the first 5 rows of a DataFrame.

tail()-

Displays the last 5 rows of a DataFrame.

shape-

Returns the number of rows and columns in the DataFrame.

ndim-

Returns the number of dimensions of the DataFrame.

size-

Returns the total number of elements in the DataFrame.

columns-

Displays the names of all columns in the DataFrame.

dtypes-

Shows the data type of each column in the DataFrame.

df["column_name"]-

Accesses an individual column from the DataFrame.

iloc[]-

Used for integer-location based indexing to access specific rows or elements.

loc[]- 

Used for label-based indexing to update or retrieve values.

df["new_column"]- 

Adds a new column to the DataFrame.

drop()-
Removes a specified column or row from the DataFrame.
